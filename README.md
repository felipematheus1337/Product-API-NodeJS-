# Product-API-NodeJS-
Uma API de cadastro de produto com:

<ul>
<li>
Autenticação JWT
</li>
<li>Upload de imagens</li>
<li>CRUD completo</li>
<li>Documentação com Swagger</li>
</ul>

<li>
NodeJS
</li>
<li>Express</li>
<li>MySQL</li>
<li>Sequelize</li>
<li>Multer</li>
<li>JSON WEB TOKEN</li>
</ul>
